# Application:
פנס OI

# Promo text:
פנס OpenIntents  עוזר לך למצוא את דרכך באפלה. פנס הצג האחורי נשמר במצב פועל כל עוד יישום זה פועל.

# Description:
פנס OI הוא יישום פשוט ויעיל שעוזר לך למצוא דרך בחשכה. פנס OI נועל את הצגת הפנס האחורי, מאיר בבירור את האיזור המקיף. עבור מכשירים עם מצלמת פלאש, פנס OI יכול גם להשתמש בפלאש כלפיד בהיר כדי להאיר את החדר. הצבע של הפנס האחורי הוא ניתן לעיצוב עם בוחר צבעים OI. 

באפליקציה זאת אין פרסומות והיא לא דורשת אישור אינטרנט. 

קוד המקור של אפליקציה חינמית ובעלת קוד פתוח זמין אצל: 
http://code.google.com/p/openintents

לרשימה המלאה של שינויים ולרשימה של שאלות נפוצות אנא לכו אל: 
http://www.openintents.org

אתם יכולים לעזור לשפר את התרגום בשפתכם ב-Launchpad: 
https://translations.launchpad.net/openintents/trunk

מאפיינים: 
* נעל את הצגת הפנס האחורי לפועל.
* נעל פלאש המצלמה במצב פועל.
* הצבע של הפנס האחורי ניתן לעיצוב

הרחבות זמינות עבור "פנס OI": 
* הראה מידע על הרישיון עם "אודות OI".
* שנה את צבע הפנס האחורי עם "בוחר צבעים OI"
