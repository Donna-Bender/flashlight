# Application:
OI Latarka

# Promo text:
Pozostaw lampę błyskową lub podświetlenie ekranu włączone używając OI Latarka.

# Description:
OI Flashlight to prosta i skuteczna aplikacja, która pomaga znaleźć drogę w ciemnościach. OI Flashlight blokuje podświetlanie, jasno oświetla okolicę. Dla urządzeń z lampą błyskową aparatu, Ol Flashlight może również korzystać z lampy błyskowej jako jasna latarka, aby rozjaśnić pokój. Kolor podświetlenia jest w pełni konfigurowalny z OI Próbnik Kolorów. 

Ta aplikacja jest darmowa i nie wymaga dostępu do internetu. 

Kod źródłowy tej darmowej i open-source'owej aplikacji jest dostępny pod adresem: 
http://code.google.com/p/openintents

Aby zobaczyć pełną listę zmian oraz FAQ, odwiedź: 
http://www.openintents.org

Możesz poprawić tłumaczenie na Twój język: 
https://translations.launchpad.net/openintents/trunk

Cechy: 
* Zablokuj podświetlenie.
* Zablokuj lampę błyskową.
* Kolor podświetlenia można dostosować.

Dostępne rozszerzenia dla "OI Latarka": 
* Pokaż informacje licencyjne za pomocą "OI About".
* Zmień kolor podświetlania za pomocą "OI Próbnik Kolorów".

Aplikacje działające z "OI Latarka": 
* Aktualizuj za pomocą "OI Uptade".
