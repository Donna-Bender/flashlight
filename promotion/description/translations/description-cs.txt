# Application:
OI Baterka

# Promo text:
Uchovejte blesk nebo podsvícení zapnutý s OI Baterkou.

# Description:
OI Baterka je jednoduchá a efektivní aplikace, s níž najdete svou cestu tmou. OI Baterka zapne podsvícení displeje, jasně osvětlujíc okolní prostředí. Na zařízeních s bleskem fotoaparátu umí OI Baterka použít i ten jako svítilnu pro rozjasnění místnosti. Barva podsvícení je nastavitelná pomocí OI Výběru barev. 

Tato aplikace je bez reklam a nevyžaduje přístup k internetu. 

Zdrojové kód této open source aplikace je dostupný na stránkách: 
http://code.google.com/p/openintents

Pro úplný seznam změn a seznam často kladených dotazů navštivte prosím: 
http://www.openintents.org

Můžete také pomoci zlepšit překlad do Vašeho jazyka na Launchpadu: 
https://translations.launchpad.net/openintents/trunk

Vlastnosti: 
* Nastaví podsvícení displeje na maximum a neumožní mu se vypnout.
* Zapne blesk fotoaparátu.
* Barva podsvícení je nastavitelná.

Dostupná rozšíření pro OI Baterku: 
* Zobrazit licenční infomace s OI Infem.
* Změnit barvu podsvícení OI Výběrem barev.

Aplikace, které pracují s OI Baterkou: 
* Aktualizovat s OI Update.
