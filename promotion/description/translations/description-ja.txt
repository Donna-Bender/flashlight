# Application:
OI フラッシュライト

# Promo text:
OI フラッシュライト でカメラフラッシュかバックライトを常に点灯させることができます。

# Description:
OI フラッシュライトは暗闇の中であなたを助けるシンプルで効果的なアプリケーションです。 OI フラッシュライトはディスプレーバックライトをオンの状態にロックし、周囲を明るく照らします。 OI フラッシュライト、カメラフラッシュがついているデバイスではフラッシュを使って部屋を明るくするトーチになることができます。 バックライトのカラーは OI カラーピッカー で完全にカスタマイズすることができます。 

このアプリケーションは広告を含まず、インターネット許可を必要としません。 

この無料でオープンソースのアプリケーションのソースコードは以下の場所から取得できます： 
http://code.google.com/p/openintents

すべての変更リストや質問リストを見るには： 
http://www.openintents.org

あなたもこの言語の翻訳を Launchpad で改善することができます： 
https://translations.launchpad.net/openintents/trunk

機能： 
* ディスプレーバックライトをオンにロックする。
* カメラのフラッシュをオンにロックする。
* バックライトのカラーは変更することができます。

"OI フラッシュライト" で利用可能な拡張子： 
* "OI 情報" のライセンス情報を表示する。
* "OI カラーピッカー" でバックライトの色を変更する。

"OI フラッシュライト" と動作するアプリケーション： 
* "OI アップデート" でアップデート。

キーワード: 
* フラッシュライト、ライト、バックライト
* OI, OpenIntents, Open Intents
