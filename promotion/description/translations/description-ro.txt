# Application:
Lanternă OI

# Promo text:
Keep the camera flash or the backlight turned on with OI Flashlight.

# Description:
OI Lanternă este o aplicaţie simplă şi eficientă, care vă ajută să găsiţi o cale în întuneric. OI Lanternă pastrează iluminarea din spate a ecranului pornită, iluminând în mod clar zona înconjurătoare. Pentru dispozitivele cu bliţ, lanterna OI poate utiliza de asemenea blițul ca o torță luminoasă pentru a lumina camera. Culoarea iluminării din spate este personalizabilă cu OI Color Picker. 

Această aplicație nu conține reclame și nu necesită permisiunea de internet.

* Păstrați iluminarea din spate a afișajului pornită.
* Păstrați blițul pornit.
* Culoarea iluminării din spate este personalizabilă.
