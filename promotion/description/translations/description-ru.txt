# Application:
OI Flashlight

# Promo text:
OpenIntents Flashlight поможет Вам найти дорогу в темноте. Пока запущено приложение - подсветка экрана остается включенной.

# Description:
OI Flashlight это простое и эффективное приложение которое помогает найти дорогу в темноте. OI Flashlight включает подсветку, освещает окрестности Для устройств со вспышкой OI Flashlight можно использовать вспышку, как яркий факел, чтобы осветить комнату Цвет подсветки настраивается с помощью OI Color Picker 

Это приложение является бесплатным и не требует постоянного подключения к Интернет. 

Исходный код этой программы является свободным и открытым. Доступен по адресу: 
http://code.google.com/p/openintents

С полным списком изменений и списком часто задаваемых вопросов можно ознакомиться на: 
http://www.openintents.org

Вы можете помочь улучшить перевод прилоежния на ваш язык, посетите Launchpad: 
https://translations.launchpad.net/openintents/trunk

Возможности: 
* Включить подсветку экрана
* Включить вспышку
* Цвет подсветки настраивается

Доступные дополнения для "OI Flashlight": 
* Показать информацию о лицензии с помощью "OI About".
* Изменить цвет подсветки с помощью OI Color Picker

Приложения работающие с "OI Flashlight": 
* Обновить с помощью "OI Update".
