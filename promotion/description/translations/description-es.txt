# Application:
OI Flashlight

# Promo text:
OpenIntents FlashLight te ayuda a encontrar tu camino en la oscuridad. La luz de fondo de la pantalla permanece encendida mientras la aplicación esté arrancada.

# Description:
OI Flashlight es una aplicación sencilla y eficaz que le ayuda a encontrar el camino en la oscuridad. OI Lightflash enciende la retroiluminación de la pantalla, iluminando el área circundante. Para los dispositivos con flash de cámara, OI Flashlight puede emplearlo como linterna para iluminar la habitación. El color de fondo es completamente personalizable con OI Color Picker. 

Esta aplicación no tiene anuncios y no requiere permisos de internet. 

El código fuente de esta aplicación libre y de código abierto está disponible en: 
http://code.google.com/p/openintents

Para la lista completa de cambios y la lista de preguntas frecuentes, por favor visite: 
http://www.openintents.org

Usted puede ayudar a mejorar la traducción de su idioma en Launchpad: 
https://translations.launchpad.net/openintents/trunk

Características: 
* Bloqueo de la retroiluminación de la pantalla.
* Bloqueo del flash de la cámara.
* El color de fondo es personalizable.

Extensiones disponibles para "OI Flashlight": 
* Mostrar la información de licencia en "OI About".
* Cambiar el color de fondo con "OI Color Picker".

Aplicaciones que funcionan con "OI Flashlight": 
* Actualizar con "OI Update".
