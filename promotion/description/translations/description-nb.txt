# Application:
OI Lykt

# Promo text:
Hold kamerablitsen eller baklyset skrudd på med OI Lykt.

# Description:
OI Lykt er en enkel og effektiv applikasjon som hjelper deg finne veien i mørket. OI Lykt låser skjermens baklys på, tydelig opplysende for det omkringliggende området. For enheter med en kamerablits, OI Lykt kan også bruke blitsen som en lys lykt for å lyse opp rommet. Fargen på baklyset er helt egendefinerbart med OI Fargevelger. 

Denne applikasjonen er reklamefri og trenger ikke inter 

Kildekoden til denne gratis applikasjonen med åpen kildekode er tilgjengelig hos: 
http://code.google.com/p/openintents

For den komplette listen med endringer og en liste over vanlige spørsmål, vennligst besøk: 
http://www.openintents.org

Du kan hjelpe til med å forbedre oversettelsen 
https://translations.launchpad.net/openintents/trunk

Funksjoner: 
* Låser baklyset på.
* Lås kamerablits på.
* Fargen på baklyset er egendefinerbart.

Tilgjengelige utvidelser for "OI Lykt": 
* Vis lisensinformasjonen med "OI Info".
* Endre fargen på baklyset med "OI Fargevelger".

Applikasjoner som fungerer med "OI Lykt": 
* Oppdater med "OI Oppdater".
